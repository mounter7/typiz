# Typiz

- Open the terminal in linux or mac
    ### or
- Open the command prompt(cmd) in windows and just type,
```
pip install -r requirements.txt
```
```
typiz -m <message> -n <number of times> -t <delay time>
```
for example:
```
typiz -m "Hey guys!" -n 100 -t 5
```
